# CodeAlpha Internship - Task 2: Stock Portfolio Tracker

This repository contains **Task 2: Stock Portfolio Tracker**, developed as a part of my Python Programming Internship at CodeAlpha.

## 📌 Project Overview
The objective of this project is to build a simple, console-based stock tracking application that allows users to input their stock names and quantities to calculate their total investment value instantly

The application utilizes core programming fundamentals and data collections to process tracking metrics without relying on heavy external database frameworks

## 🛠️ Project Features
* **Baseline Stock Database:** Uses a hardcoded Python dictionary to define standard stock baselines and prices (e.g., AAPL, TSLA)
* **Interactive Terminal Input:** Captures user stock preferences and automatically processes input data into uppercase formatting to ensure exact matches
* **Input Protection Workflow:** Incorporates safe numeric validation (`.isdigit()`) to verify input quantities, ensuring the script handles errors cleanly without crashing.
* **Automated File Export:** Utilizes basic Python file handling tools (`open()`, `write()`, `close()`) to save and log the final portfolio calculations directly into a local text file (`my_portfolio.txt`)

## 💻 Key Python Concepts Covered
* **Dictionaries:** Mapping immutable keys to fixed stock pricing streams.
* **Control Flow & Predicates:** Evaluating system listings using precise `if-else` blocks.
* **Type Casting:** Converting operational string inputs securely into mathematical integers
* **File I/O Stream Handling:** Overwriting and creating persistent reports locally
