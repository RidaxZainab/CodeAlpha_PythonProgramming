# CodeAlpha Python Programming Internship

Welcome to my Python programming portfolio for the **CodeAlpha Internship Program**. This repository serves as a centralized hub for all the tasks completed and explored during this hands-on learning experience.

## 📌 About the Internship
CodeAlpha is a leading software development company that empowers students to master Python fundamentals, data structures, and OOP concepts while working on real-world projects. 

## 🛠️ Tech Stack & Key Concepts Used
* **Language:** Python
* **Core Concepts:** Dictionaries, Strings, Lists, Conditional Statements (if-elif-else), Loops, and Functions.
* **Libraries & Modules:** `random`, `os`, `shutil`, `re`, `requests`.
* **Advanced Application:** System Input/Output and File Handling (Reading/Writing data).

## 📂 Project Tasks Overview
According to the internship criteria, participants are required to complete a minimum of two or three tasks from the official domain list:

### 🔹 Task 1: Hangman Game
* **Goal:** A simple, text-based word-guessing game.
* **Features:** * Uses a predefined list of 5 words randomly selected using the `random` module.
  * Limits the user to a maximum of 6 incorrect guesses.
  * Implements interactive loop-based console input without external graphics or audio.

### 🔹 Task 2: Stock Portfolio Tracker
* **Goal:** A console-based application that tracks total investments based on user inputs.
* **Features:** * Uses a hardcoded dictionary for tracking stock baselines (e.g., AAPL, TSLA).
  * Takes user input for stock names and shares owned.
  * Performs basic arithmetic operations to calculate total investment value.
  * Automatically exports and saves the final calculation summary into a local text file.

### 🔹 Task 3: Task Automation with Python Scripts
* **Goal:** An automation script to optimize small, repetitive daily tasks.
* **Features:** * Implements targeted file/data manipulation using core Python libraries (`os`, `shutil`, `re`, or `requests`).
  * *Options include:* Batch moving `.jpg` files between folders, parsing and extracting email addresses from a text file, or scraping specific webpage titles.

### 🔹 Task 4: Basic Chatbot
* **Goal:** A simple, rule-based conversational assistant.
* **Features:** * Handles standard user inputs such as greetings and casual questions (e.g., "hello", "how are you").
  * Delivers instant, pre-defined smart responses utilizing structured `if-elif` logic and functional loops.
